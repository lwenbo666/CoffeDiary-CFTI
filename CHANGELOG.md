# 版本更新日志

## v0.1.2.4 (2026-05-22)

### 🔧 优化
- **月历收起动画重写**：解决收起时"先收下面、再收上面"和"动画结束突然跳变"两段式卡顿
- 收起动画改为单段连续过渡：月导航栏与日历内容并行收起，周视图 DOM 提前静默替换
- **月历左右切换动画**：新增 `navigateMonth()` 统一动画函数，按钮点击和触摸滑动共用滑出→换月→滑入动画，视觉流畅
- **HTML/JS 分离**：`main.html` 和 `menu.html` 的 JavaScript 代码拆分为独立 `main.js` / `menu.js`，通过 `defer` 加载
- **记录排序优化**：咖啡记录列表按日期降序排列（新记录显示在最上面）
- **柱状图优化**：一周饮用节律柱状图使用渐变色渲染、添加数值标签、优化轴标签样式和动画参数
- CSS `will-change: max-height` + `contain: layout style` 启用 GPU 合成层优化
- 收起过渡时间缩短至 0.35s，使用 `transitionend` 事件精确控制动画结束
- 双 `rAF` 确保锁定帧完整渲染后再启动过渡，消除 `none→数值` 假动画
- 七日视图单元格高度从 100px 缩减至 64px，去除多余 padding-bottom 空白
- 折叠高度固定为 108px，贴纸靠 `overflow: visible` 自然溢出
- 贴纸外框取消圆角（`border-radius: 16px → 0`）

### 🚀 技术
- 移除 `CoffeeSegmenter.kt` 和 `download_model.py`（抠图功能相关文件）

## v0.1.2.3 (2026-05-21)

### ✨ 新增
- 常喝咖啡 / 所有记录点击贴纸：取消全屏图片查看器，改为从左侧原位丝滑弹出至屏幕中央
- 弹出贴纸为直角方形，白边框，点击空白处或按 Esc 缩回原位
- 弹出遮罩背景为透明，周围不变暗
- **咖啡杯轮廓抠图**：拍照界面新增「✂️ 抠图」开关，启用后使用 TensorFlow Lite + BRIA RMBG-1.4 模型自动识别咖啡杯轮廓并移除背景，生成透明背景贴纸
- 抠图处理时显示加载动画，处理完毕后进入命名流程

### 🔧 优化
- 左侧小贴纸取消圆角，改为直角方形（`border-radius` 移除）
- 动画性能优化：`fadeInUp` 替换为 `translate3d` + `will-change` + `backface-visibility`，启用 GPU 合成层，消除 repaint 掉帧
- 动画调度优化：统计页 3 个 Chart.js 实例改为懒初始化（首次切换到统计页时才创建），降低首页主线程负载
- 动画调度优化：日历 DOM 构建推迟至双 `rAF`，避免与 `fadeInUp` 抢占帧时间片
- 动画时序修复：`fadeInUp` 默认 `animation-play-state: paused`，Splash 消失后由 Kotlin `evaluateJavascript` 注入 `body.app-ready` 才启动，解决动画在 Splash 遮挡期间跑完导致的假性掉帧
- 页面切换过渡时长 `0.28s` → `0.35s`，视觉更从容

### 🚀 技术
- JS 兜底超时：`body.app-ready` 5 秒未注入时自动触发动画，防止永远卡住
- TFLite 集成：RMBG-1.4 背景移除模型，GPU → NNAPI → CPU 三级委托回退
- `CoffeeSegmenter.kt`：原生 Kotlin 分割推理，`segmentCoffeeCup()` JS 桥接
- `download_model.py`：从 HuggingFace 下载 ONNX 并转为 TFLite，放入 assets

## v0.1.2.2 (2026-05-21)

### 🔧 优化
- 咖啡页、统计页顶部间距从 `6rem` 缩减至 `2.5rem`（约 40px），视觉更紧凑
- 菜单页头部间距与咖啡页/统计页统一为 40px
- 菜单页改为左右翻页浏览，每页展示一张咖啡配方卡片，使用 CSS `scroll-snap` 实现平滑翻页
- 菜单页添加页面指示器（圆点），直观显示当前页码和总页数
- 菜单页隐藏所有滚动条，界面更干净
- 性能优化：`saveCoffeeRecord`/`deleteCoffeeRecord` 等操作使用 `rAF` 防抖合并 UI 更新，避免单次操作触发 4 次 DOM 重写
- 性能优化：Release 构建启用 R8 混淆 + 资源压缩（APK 体积缩减 30%~60%）
- 性能优化：Splash 动画缓存缩放 Bitmap，消除每帧创建临时 Bitmap（约 132 次创建 → 1 次）
- 性能优化：菜单页 `escapeHtml` 从 DOM 方式改为字符串替换，减少临时元素创建
- 性能优化：菜单页表单元素移除冗余 `backdrop-filter`（已由模态框毛玻璃覆盖）
- 菜单页 `localStorage` 存储增加容量检查，超出限制时自动压缩图片后重新保存
