# 版本更新日志

## v0.1.2 (2026-05-21)

### ✨ 新增
- 统计页不满 7 天时 CFTI 区域不再隐藏，改为显示"没有记录七天无法生成"提示

### 🔧 优化
- 温度偏好和糖度选择色环（环形图）禁用点击交互，避免误触
- 贴纸照片分辨率从 80×80 提升至 400×400（25×），放大查看清晰不模糊
- 贴纸比例调整为 5:6 竖版圆角矩形
- 日历网格间距缩小，内边距优化，格子更宽
- 贴纸容器尺寸调大，视觉更饱满

### 🚀 技术
- 贴纸放大改用直接渲染尺寸替代 CSS `transform: scale()`，消除像素拉伸模糊
- VersionCode: 2 → 3
